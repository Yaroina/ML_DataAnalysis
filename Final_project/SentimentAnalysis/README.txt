Скриншоты работы приложения находятся в папке Screenshots.
Логика следующая: скриншот Example_1 - реальный отзыв с МВидео, App_1 - результат работы анализатора для него.

Основной код приложения находится в папке app.

Запуск приложения:

1. Запустить run.py
2. Дождаться, когда будет готов классификатор. (5-6 сек.)
3. Открыть браузер
4. В браузере ввести http://localhost:5000
5. Окажемся на домашней странице. Перейти на страницу Анализатор
для анализа тональности отзывов.
